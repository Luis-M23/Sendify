## Environment Variables
Para correr este proyecto, necesitaras agregar las siguientes variables de entorno en el archivo .env

que debes de crear al momento de clonar el repositorio.

`NEXT_PUBLIC_SUPABASE_URL`

`NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY`

`SUPABASE_SERVICE_ROLE_KEY`


Estas variables las optines al momento de crear tu proyecto en supabase.
